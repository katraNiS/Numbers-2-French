# Numbers 2 French 🇫🇷

A simple web app that helps you learn French numbers. Generate a random number, then see its French translation. Perfect for beginners who want to practice numbers from 0 to 999.

## Features
- Random number generator with min/max range filter
- Translates numbers from 0–999 into French
- Clean, mobile-friendly design

## Upcoming Features
- Numbers up to 1,000,000
- Reverse mode: see the French word first and guess the number
- Quiz mode: type your answer and check if it's correct
- Greek language support

## Tech Stack
- HTML
- CSS
- JavaScript
